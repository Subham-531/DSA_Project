About Linked List

A Linked List is a linear data structure where elements are stored in nodes. Each node contains data and a pointer to the next node. Unlike arrays, linked lists allow efficient insertion and deletion at any position.

Insert: O(1) / O(n) Delete: O(1) / O(n) Search: O(n)

About Stack

A Stack follows the Last In, First Out (LIFO) principle. Think of a stack of plates — you can only add or remove the top plate. Stacks are used in function calls, undo mechanisms, and expression evaluation.

Push: O(1) Pop: O(1) Peek: O(1)

About Queue

A Queue follows the First In, First Out (FIFO) principle. Like a line at a ticket counter — the first person to arrive is the first to be served. Queues are used in scheduling, BFS, and buffering.

Enqueue: O(1) Dequeue: O(1) Peek: O(1)
